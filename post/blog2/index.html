Redis安装&部署&远程连接（Linux）
@ CodeV · Jul 8, 2021 · 1 min read · Update at Jul 8, 2021

准备Redis安装包，并解压

  • 进入redis官方网站下载地址，选择自己合适的版本,并放在root目录下
  • 在 /usr/local/ 下创建 redis 文件夹并进入
cd /usr/local/
mkdir redis
cd redis
  • 将redis安装包解压到 /usr/local/redis 中
tar -zxvf /root/redis-5.0.8.tar.gz -C ./
  • 解压完之后，/usr/local/redis目录中会出现一个redis-5.0.8目录

编译并安装

cdredis-5.0.8/
make && make install

PS: 如果中途报错，可以去检查一下gcc编译器是否安装，rpm -q gcc查看，如果编译出错，请使用make clean清除临时文件。之后，找到出错的原因，解决问题后再来重新安装

将Redis安装为系统服务并后台启动

  • 进入utils目录，并执行如下命令
cd utils/
./install_server.sh

PS: 配置默认就行

  • 查看redis服务的启动情况
systemctl status redis_6379.service

  • 启动自带的redis-cli客户端，测试通过

设置远程链接

  • 编辑redis配置文件
vim /etc/redis/6379.conf
在 69 行的位置将 bind 127.0.0.1 注释掉

  • 88行protected-mode改为yes
  • 找到507行的requirepass foobared去掉注释并将foobared改为自己的密码 **PS：**行号设置，请看这篇文章，其中配置文件的位置，我的电脑是在 /etc 文件夹中

防火墙设置并重启

  • 在防火墙设置中将6379端口打开，因为我自己用的是iptables配置命令如下
vim /etc/sysconfig/iptables
-A INPUT -m state --state NEW -m tcp -p tcp --dport 6379 -j ACCEPT


重启防火墙：
service iptables restart
重启redis:
systemctl restart redis_6379.service
